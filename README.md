# 🏋️ Трекер Привычек — Telegram Бот

Бот для ежедневного трекинга привычек с напоминаниями, прогресс-барами и стриками.

## 🚀 Быстрый старт

### 1. Получи токен бота
1. Открой Telegram, найди [@BotFather](https://t.me/botfather)
2. Напиши `/newbot`
3. Дай боту имя и юзернейм
4. Скопируй токен (выглядит как `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

### 2. Вставь токен
Открой файл `.env` и замени плейсхолдер своим токеном:
```
BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

### 3. Запусти бота
```bash
python bot.py
```

---

## 📋 Что умеет бот

| Функция | Описание |
|---------|---------|
| ➕ Добавить привычку | Количественная (с числом) или Да/Нет |
| ✏️ Отметить | Логируешь сколько сделал, бот прибавляет к уже записанному |
| 📊 Статистика | За сегодня / неделю / месяц |
| 🔥 Стрики | Серии дней подряд с красивыми эмодзи |
| 📈 Прогресс-бар | `▓▓▓▓▓░░░░░ 50%` |
| 🔔 Напоминания | Каждый день в заданное время |

## 📁 Структура проекта

```
бот/
├── bot.py              ← Точка входа (запускай это)
├── config.py           ← Конфиг
├── .env                ← Токен (не коммить в git!)
├── requirements.txt    ← Зависимости
├── habits.db           ← База данных (создаётся автоматически)
├── database/db.py      ← Работа с SQLite
├── handlers/
│   ├── start.py        ← /start и главное меню
│   ├── habits.py       ← Добавление/удаление привычек
│   ├── logging.py      ← Отметка выполнения
│   └── stats.py        ← Статистика + настройка напоминаний
├── keyboards/inline.py ← Все inline-кнопки
├── scheduler/reminders.py ← Ежедневные напоминания
└── utils/formatters.py ← Прогресс-бары и форматирование
```

## ⚠️ Важно

- Файл `.env` с токеном **не выкладывай** в публичный репозиторий
- База данных `habits.db` создаётся автоматически при первом запуске
- По умолчанию напоминания в **09:00** — меняешь в боте кнопкой "⏰ Напоминание"
